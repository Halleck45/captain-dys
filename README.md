# Captain Dys

[captaindys.com](https://captaindys.com/). A web editor for (french for the moment) dyslexic child.

![preview](./docs/preview.png)

## Installation

```bash
yarn install
npm run build
```

## Contributing

In order to work locally, run:

```bash
yarn install
npm run serve --mode=development
```
